1.2. Текущая функциональность
Торговля на Bybit (CRYPTO_BOT), Тинькофф (TINKOFF_BOT), Мосбирже (MOEX_BOT)

Multi-coin (один бот → много символов)

Автооптимизация параметров (Optuna)

Градуированные триггеры (0/1/2) с защитой от ложных срабатываний

Market Regime Filter (ADX + ATR, приоритет HIGH_VOLATILITY)

Risk Manager + Volatility Guard (восстановление после Halt)

3 стратегии (MA Crossover, Bollinger, SuperTrend)

Out-of-sample проверка (overfit_ratio)

Telegram управление с префиксами (crypto_, tinkoff_, moex_)

reload_flag (применение параметров без перезапуска)

Инфраструктура тестирования (L1/L2/L3, pytest, ручной чек-лист)

1.3. Текущие ограничения
Старые single-coin боты ещё работают (технический долг)

Нет автоматических еженедельных отчётов (ручной режим)

Нет накопленной статистики реальной торговли

Prediction Module не реализован (отложен до V2.0)

2. Ближайшие этапы (V1 — Стабилизация и реальная торговля)
Этап 1: Демо-торговля (Спринт №7)
Тип: Эксплуатационный этап (не разработка)

Задача	Длительность	Ответственный
Запуск демо-торговли на всех трёх ботах	1 день	Архитектор (по инструкции Проектировщика)
Ежедневный мониторинг логов и метрик	7-14 дней	Архитектор + Проектировщик (анализ)
Настройка порогов триггеров под реальные данные	По мере необходимости	Проектировщик (после консультации с Консультантом)
Сбор статистики, выявление проблем	14 дней	Проектировщик
Еженедельные отчёты (ручной режим)	Каждый понедельник	Проектировщик (инструкция) → Архитектор (отправка)
Критерий перехода к следующему этапу: Система стабильно работает на демо-счетах 2 недели без критических ошибок.

Ориентировочная длительность: 2-3 недели

Этап 2: Реальная торговля минимальным капиталом (Спринт №8)
Тип: Эксплуатационный этап

Задача	Длительность	Ответственный
Переключение ботов на реальные счета (минимальный капитал)	1 день	Архитектор
Ежедневный мониторинг реальных сделок	14 дней	Архитектор + Проектировщик
Анализ проскальзывания, комиссий, задержек	7 дней	Проектировщик + Консультант
Корректировка риск-параметров	По необходимости	Проектировщик
Еженедельные отчёты по реальной торговле	Каждый понедельник	Проектировщик → Архитектор
Критерий перехода к следующему этапу: Система показывает стабильную (пусть и небольшую) положительную динамику или околонулевую с контролируемыми просадками.

Ориентировочная длительность: 3-4 недели

Этап 3: Эксплуатация и накопление данных (после Спринта №8)
Тип: Эксплуатационный этап (без активной разработки)

Задача	Длительность	Ответственный
Торговля на реальных счетах без новых фич	3-4 недели	Архитектор (запуск), Проектировщик (мониторинг)
Сбор метрик для анализа эффективности	Весь период	Проектировщик
Выявление слабых мест в эксплуатации	Весь период	Архитектор + Проектировщик
Еженедельные отчёты с рекомендациями	Каждый понедельник	Проектировщик → Архитектор
Критерий перехода к V2: Накоплена достаточная статистика (минимум 100 сделок на каждого бота) для принятия решения о дальнейшем развитии.

Ориентировочная длительность: 3-4 недели

3. Следующие этапы (V1.1 — Оптимизация и миграция)
Этап 4: Миграция single-coin ботов (Спринт №9)
Тип: Разработка + эксплуатация

Задача	Длительность	Ответственный
Перенос символов старых ботов в CRYPTO_BOT	3 дня	Разработчик
Тестирование (L2/L3)	2 дня	Тестировщик
Поэтапное отключение старых ботов	1 неделя	Архитектор
Удаление кода старых ботов (архивация)	1 день	Архитектор
Критерий завершения: Все single-coin боты отключены, их символы торгуются через CRYPTO_BOT.

Ориентировочная длительность: 1.5-2 недели

Этап 5: Автоматизация отчётов (задача из бэклога)
Тип: Разработка

Задача	Длительность	Ответственный
Создание скрипта scripts/weekly_report.py	2 дня	Разработчик
Интеграция с Telegram Bot API	1 день	Разработчик
Настройка cron для еженедельного запуска	1 день	Архитектор
Тестирование	1 день	Тестировщик
Ориентировочная длительность: 1 неделя (может быть включена в Спринт №9 или отдельным микро-спринтом)

Этап 6: Расширение инструментов (бэклог)
Тип: Разработка (по мере приоритизации)

Задача	Приоритет	Ориентировочная длительность
Новые монеты на CRYPTO_BOT (AVAX, DOT, MATIC, LINK)	Средний	3-5 дней
Новые акции на TINKOFF_BOT (префы, second tier)	Низкий	3-5 дней
Новые фьючерсы на MOEX_BOT (золото, нефть, евро, юань)	Низкий	3-5 дней
Решение о включении принимает Архитектор на основе накопленной статистики.
